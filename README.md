# 📝 TP INF222 : API RESTful de Gestion de Blog
**Université de Yaoundé 1 - Niveau 2 Informatique**

Ce projet consiste en la mise en place d'une API backend complète permettant la gestion d'articles de blog. Il intègre une documentation interactive via **Swagger UI** pour faciliter les tests des différents points de terminaison (endpoints).

---

## 🚀 Fonctionnalités
L'API supporte toutes les opérations CRUD de base ainsi que des filtres avancés :
* **Affichage** : Liste complète de tous les articles.
* **Création** : Ajout de nouveaux articles via un formulaire structuré.
* **Recherche spécifique** : Récupération d'un article par son identifiant unique (ID).
* **Filtrage temporel** : Recherche d'articles par date de publication (AAAA-MM-JJ).
* **Moteur de recherche** : Recherche par mot-clé dans les titres des articles.
* **Suppression** : Retrait d'articles existants.

---

## 🛠️ Technologies Utilisées
* **Node.js** & **Express** : Environnement d'exécution et framework backend.
* **Swagger UI** : Interface de documentation interactive (OpenAPI 3.0).
* **JavaScript (ES6+)** : Logique serveur et manipulation de données.
